# 💻 تشغيل نظام موارد بشرية ورواتب مجموعة عرفه على جهازك

## ✅ المتطلبات (مرة واحدة)

### 1. تثبيت Node.js (لو غير مثبت)
- اذهب إلى https://nodejs.org
- حمّل إصدار **LTS** (الإصدار 20 أو أحدث)
- ثبّته بشكل عادي (Next → Next → Finish)

### 2. تثبيت Bun (أسرع من npm — مُفضّل)
- افتح **Command Prompt** أو **PowerShell**
- شغّل:
```bash
npm install -g bun
```

أو استخدم npm مباشرة (لا مشكلة).

---

## 🚀 خطوات التشغيل

### الخطوة 1: تحميل ملفات المشروع
- حمّل ملف `arafa-hr-system.zip` من قسم الإعدادات
- أو من الرابط المباشر: `/api/backup/project`

### الخطوة 2: فك ضغط الملف
- انقر بزر الفأرة الأيمن على الـ zip
- اختر **"Extract All"** (استخراج الكل)
- اختر مجلداً مثل: `C:\arafa-hr-system`

### الخطوة 3: فتح موجه الأوامر في المجلد
- افتح **Command Prompt** أو **PowerShell**
- انتقل لمجلد المشروع:
```bash
cd C:\arafa-hr-system
```

### الخطوة 4: تثبيت الحزم (مرة واحدة)
```bash
bun install
```
أو لو تستخدم npm:
```bash
npm install
```

انتظر حتى ينتهي (قد يأخذ 2-5 دقائق).

### الخطوة 5: إعداد قاعدة البيانات (مرة واحدة)
```bash
bun run db:push
```
أو:
```bash
npx prisma db push
```

هذا ينشئ قاعدة البيانات ويضيف المستخدم الافتراضي (admin).

### الخطوة 6: إعادة تعيين كلمة المرور (مهم!)
بعد تثبيت الحزم ودفع قاعدة البيانات، شغّل هذا الأمر لإعادة تعيين كلمات المرور:
```bash
bun run reset-password
```
أو:
```bash
node scripts/reset-password.js
```

هذا سيضمن أن كلمات المرور تعمل بشكل صحيح على جهازك.

### الخطوة 7: تشغيل البرنامج
```bash
bun run dev
```
أو:
```bash
npm run dev
```

### الخطوة 8: فتح البرنامج في المتصفح
افتح المتصفح واذهب إلى:
```
http://localhost:3000
```

## ❓ حل المشاكل الشائعة

### المشكلة: "bun not found"
**الحل**: استخدم `npm` بدلاً من `bun`:
```bash
npm install
npm run dev
```

### المشكلة: "port 3000 already in use"
**الحل**: استخدم منفذ آخر:
```bash
bun run dev -- -p 3001
```
ثم افتح: `http://localhost:3001`

### المشكلة: "DATABASE_URL not set"
**الحل**: تأكد أن ملف `.env` موجود ويحتوي:
```
DATABASE_URL=file:./db/custom.db
JWT_SECRET=arafa-hr-secret-key-2025-very-secure
```

### المشكلة: خطأ في Prisma
**الحل**: أعد توليد Prisma:
```bash
npx prisma generate
npx prisma db push
```

### المشكلة: الشاشة بيضاء
**الحل**: امسح مجلد `.next` وأعد التشغيل:
```bash
rm -rf .next
bun run dev
```
على Windows:
```bash
rmdir /s /q .next
bun run dev
```
